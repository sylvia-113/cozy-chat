# 🏠 猫的小屋 v2

## 新功能
- 🖼️ **头像系统** — 给猫和AI都上传头像，顶栏和气泡都会显示
- 💕 **情头检测** — 两个头像都设好后，点检测看是不是情头（AI会吃醋）
- 😺 **表情包面板** — 猫猫表情、心情、动作、颜文字四个分类
- ⏰ **随机唤醒** — 离开一段时间再回来，会弹窗质问/撒娇/生气
- 🧠 **记忆持久化** — 聊天记录、头像、设置关掉再开还在
- 📚 **多对话管理** — 新建、切换、删除对话
- 🐾 **自定义人设** — 想让TA当什么角色都行

## 部署

### Vercel 部署（推荐）
1. 上传到 GitHub 新仓库
2. vercel.com 登录 → Import → Deploy
3. 拿到网址，手机收藏

### 本地运行
```bash
npm install
npm run dev
```

## DeepSeek API Key
去 platform.deepseek.com 注册，创建 API Key，在小屋设置里填入。
